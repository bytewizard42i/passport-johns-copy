# Methodology

This document outlines the methodology for investigating the desired user and developer experience for a successful blockchain.

We notably care about the following personas:
- Users: should be able to, with a single QR code scan that takes a user from zero state to a named, credentialed, recoverable account in ~60 seconds
- Developers: devs should be able to write, deploy, and distribute (including attaching metadata like names and associated frontends) their applications.
